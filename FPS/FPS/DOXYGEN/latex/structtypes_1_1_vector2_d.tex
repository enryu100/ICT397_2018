\subsection{Detailed Description}
A struct that defines a vector in two-\/dimensional space. 

The \hyperlink{structtypes_1_1_vector2_d}{Vector2\+D} struct is merely a data type that represents a vector in two-\/dimensional space, providing a few functions to assist with spatial calculations. At present, it does not appear to be used.

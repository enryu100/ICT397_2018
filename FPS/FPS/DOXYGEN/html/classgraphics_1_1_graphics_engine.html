Detailed Description

A class that renders objects in a game to the screen.

The GraphicsEngine class is responsible for displaying the terrain and all of the in- game objects with a heightfield and a set of models associated with said objects. Presently, it is also responsible for polling events due to the nature of SDL, though this is something that should be changed if possible.
